# Mental Relaxation App 🧘‍♂️🌿

A full-stack web application designed to help users clear their minds, manage their thoughts, and find peace.

### 🌐 Live Demo
[https://mental-relaxation.onrender.com](https://mental-relaxation.onrender.com)

### ✨ Features
- **Mind Dump**: Type out your heavy thoughts and let them go.
- **Sentiment Analysis**: Automatically detects the mood of your thoughts (positive, negative, or neutral) using a local NLP backend.
- **Personalized Responses**: Receive an uplifting, tailored quote based on the sentiment of your thoughts.
- **Immersive Moods**: Ambient backgrounds and sounds (Rain, Fireplace, Forest) that adapt to your mood.
- **Guided Breathing**: A 1-minute guided breathing exercise to center yourself.
- **Body Scan Session**: A mini-relaxation sequence to release physical tension.

### 🚀 Tech Stack
- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **NLP**: `sentiment` (Local Natural Language Processing package)

### 🛠️ How to run locally
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the local server
4. Visit `http://localhost:3000`
